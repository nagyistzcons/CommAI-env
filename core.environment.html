core.environment module¶

class core.environment.Environment(serializer, task_scheduler, scramble=False, max_reward_per_task=10)[source]¶

Bases: object

The Environment is the one that communicates with the Learner, interpreting its output and reacting to it. The interaction is governed by an ongoing task which is picked by a TaskScheduler object.

Parameters:
  • serializer – a Serializer object that translates text into binary and back.
  • task_scheduler – a TaskScheduler object that determines which task is going to be run next.
  • scramble – if True, the words outputted by the tasks are randomly scrambled.
  • max_reward_per_task – maximum amount of reward that a learner can receive for a given task.
